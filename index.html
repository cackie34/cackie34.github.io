Launch Scripts Automated sequences for reliable liftoffs

Use these staged command flows to keep boosters synced, throttle windows safe, and orientation stable.

// Launch Sequence: "Atlas-Kerbin"
SET throttle TO 1.
LOCK STEERING TO HEADING(90, 90).
STAGE.
WAIT UNTIL ALTITUDE > 10000.
SET throttle TO 0.6.
WHEN SOLIDFUEL < 0.1 THEN STAGE.
WAIT UNTIL APOAPSIS > 75000.
LOCK STEERING TO HEADING(90, 10).
// Booster Sync Macro
SET throttle TO 0.85.
FOR engine IN ENGINES {
  IF engine:THRUST < 2.0 { engine:SHUTDOWN. }
}
